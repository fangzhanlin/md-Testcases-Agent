# 常见问题

### 1. mcp-server-time

<figure><img src="../../.gitbook/assets/telegram-cloud-photo-size-5-6068931438453048569-y.jpg" alt=""><figcaption><p>报错截图</p></figcaption></figure>

**解决方案**&#x20;

在“参数”一栏填写：

```
mcp-server-time
--local-timezone
<你的标准时区，例如：Asia/Shanghai>
```
