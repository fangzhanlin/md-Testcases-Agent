# macOS

1. 首先到官网下载页面点击下载 Mac 版本，或点击下方直达

请注意下载 **自己 Mac 对应的芯片版本**
{% hint style="info" %}
如果不知道自己的 Mac 应当使用什么芯片版本：

* 点击 Mac 左上角菜单栏 
* 在展开菜单中点击关于本机
* 在弹出窗口中查看处理器信息

如果为 Intel 芯片则下载 Intel 版本安装包

如果为 Apple M\* 芯片则下载 Apple 芯片安装包
{% endhint %}

{% embed url="https://cherry-ai.com/download" %}
