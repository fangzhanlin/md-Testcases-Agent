# 显示设置

在此页面你可以设置软件的颜色主题、页面布局或者 [自定义 CSS](../../../personalization-settings/css.md) 来进行一些个性化的设置。

### 主题选择

你可以在此处设置默认的界面颜色模式（浅色模式、深色模式或跟随系统）
