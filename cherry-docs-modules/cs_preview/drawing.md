# 绘画

绘画功能目前支持 DMXAPI、TokenFlux、AiHubMix 和 [硅基流动](../../pre-basic/providers/siliconcloud.md) 的绘画模型，你可以到 [硅基流动](https://www.siliconflow.cn/) 注册一个账户 [添加到服务商](settings/providers.md) 来使用。

有关参数的疑问可以鼠标悬停在对应区域的 `?` 处查看介绍。

{% hint style="info" %}
后续将会增加更多服务商，敬请期待。
{% endhint %}
